# Trade desk screenshots

Drop screenshots of the running trade desk here, then reference them from the "The Trade Desk" section of the top-level [README](../../../README.md), for example `![Command Center](docs/ui/screenshots/command-center.png)`.

## How to capture

1. Start the app: `cd apps/trade-desk && npm run dev`
2. Open `http://localhost:3000` in a browser at roughly 1440 wide.
3. Capture two or three pages and save them here with these names so the README picks them up:
   - `command-center.png` (the command center / home)
   - `analyze.png` (a symbol analysis with a verdict)
   - `live.png` (optional: the execution desk)

Keep images under about 1 MB each (PNG). Landscape, no browser chrome, dark theme.
